# NumPDE FS 2017

## Cloning
```
git clone --recursive https://github.com/westernmagic/NumPDE.git
```

## `master` branch
Contains my solutions

### Building
```
cd seriesX
mkdir -p build
cd build
cmake ..
cmake --build .
```

### Testing
```
ctest
```

### Generating submissions
```
cmake --build . --target package_source
```
